# 微信小游戏：像素跑酷屠龙（开发中）

本仓库包含一个**原生微信小游戏 + Canvas 2D** 的跑酷动作游戏示例（横屏），并集成**开放数据域好友排行榜**。

## 目录

- `minigame/`：小游戏主域工程（在微信开发者工具中打开此目录）
  - `game.js`：入口
  - `game.json`：配置（含开放数据域目录）
  - `src/`：核心逻辑
  - `open-data-context/`：开放数据域代码（好友榜渲染）

## 运行

1. 打开微信开发者工具 → “小游戏” → 导入项目
2. 项目目录选择 `minigame/`
3. AppID 可先用测试号/游客模式（需要好友榜时建议使用真实 AppID 并开通相关能力）
